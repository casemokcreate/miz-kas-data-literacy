# 📊 Miz Kas — Data Awareness & Literacy Resource Kit

> *Empowering organisations to understand, manage, and protect their data.*

---

## 👋 About This Project

This repository is a practical, open-source Data Awareness & Literacy Resource Kit designed to help organisations of all sizes build a data-literate workforce.

Created by **Cassandra Emokpae (Miz Kas)** — Data Governance & Compliance Specialist | Qualified Solicitor (England & Wales) | GDPR • AML/KYC • Microsoft Purview

---

## 🎯 Who Is This For?

- Organisations looking to improve data literacy across teams
- HR & Training Managers building data awareness programmes
- Compliance & Governance teams embedding data best practices
- Anyone who wants to understand data governance in plain English

---

## 📦 What's Inside

| File | Description |
|------|-------------|
| `Data_Literacy_Assessment_Quiz.xlsx` | A self-assessment quiz to measure data literacy levels across your organisation |
| `Data_Awareness_Training_Guide.docx` | A step-by-step training guide for building data awareness |
| `GDPR_Awareness_Checklist.xlsx` | A practical GDPR compliance checklist for everyday use |
| `Data_Governance_Glossary.docx` | A plain English glossary of key Data Governance terms |

## 💡 Why Data Literacy Matters

> *"Data is the most valuable asset an organisation holds. But only if people know how to use it responsibly."*

Organisations with high data literacy:
- Make better, faster decisions
- Stay compliant with GDPR and regulatory requirements
- Reduce data quality incidents
- Build trust with clients and stakeholders

---

## 🛠️ How To Use This Kit

1. Start with the **Data Literacy Assessment Quiz** to identify gaps
2. Use the **Data Awareness Training Guide** to address those gaps
3. Apply the **GDPR Awareness Checklist** to your daily operations
4. Reference the **Data Governance Glossary** for key terminology
